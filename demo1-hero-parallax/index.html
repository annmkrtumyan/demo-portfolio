Motion-first front‑ends.
Crafted for 60fps & accessibility.

Scroll to see parallax layers choreographed with GSAP ScrollTrigger. Reduced‑motion users get an instant, static layout.

Approach

Layers animate only with `transform`/`opacity`. ScrollTrigger uses scrubbed timelines; everything is clamped for performance. To respect accessibility, if the device prefers reduced motion, animations are skipped.

GSAP + ScrollTrigger

Pinless parallax with scrub control and media-query aware toggling.

Tailwind

Utility-first for rapid, consistent UI and accessible focus states.

Performance

Transform-only, no layout thrash; images are GPU-friendly.
